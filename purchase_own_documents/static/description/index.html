User's Own Forms/ Documents Only
(Odoo Purchase Module)

Description:

By default in odoo there are different predefined User Groups to manage access rights of users to different functionalities like 'User own Documents only'; but this User group is only available for Sales Module and has to be customized for other applications using odoo studio or other ways of customizations.

iTech Resources has developed this application 'User Own Document Only' for Purchase Module which restrict user to access only own Purchase Orders/ RFQ based on 'Purchase Representative' assigned on these forms. The user creating the Purchase Order/ RFQ is automatically assigned as 'Purchase Representative' on the form.
