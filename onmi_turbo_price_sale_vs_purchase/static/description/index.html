TDG12#31 - Informe Toneladas/PrecioCompra vs Toneladas/PrecioVenta

Descripción:

Una vez se reciba la factura del proveedor, se vincule mediante un listado a la factura de venta. Teniendo esta relación, solo sería realizar una resta de Ingreso - Compra. Ademas, se tiene un informe de estos cálculos.
