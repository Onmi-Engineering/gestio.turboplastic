TDG12#01 - TURBO EDI Documents

Descripción:

Se sobreescribe el metodo aplicado en ir.cron de EDI documents para que no efectúe el trigger.

Configuración:

Es necesario duplicar la acción y desactivar la anterior, ya que, si no, el trigger seguirá afectado en la acción anterior y, por ese motivo, seguiría funcionando igual.
